Futuristic Waste Management System (FWMS)

The Futuristic waste management system is a mobile application that is designed to allow users to reuse the waste product in efficient way. It allows user to sell the unwanted waste products through this platform, and those around them who are looking for the product can purchase the product from them instantly by themselves. This approach not only helps the buyers to get the product they wanted, but also help the sellers to earn money.The buyers also can be able to get a product for a cheaper price compared to the price in the market. This system act as a bridge between the seller who would like to dispose a product, and the buyer who is finding for the product. The waste products will be circulating through this application until it is fully used. This application will definitely help to reduce the amount of waste generated and encourage people to reuse the waste products.

Space Rider

Space Rider is a program that was developed for Android based smartphones which supports by at least Android 4.0. This game is designed for entertainment purpose that can be played in offline mode too. Android studio is used to develop the game with libGDX game engine which supports JAVA. This game is a basic game with the concept of riding the motorbike in the space by avoiding the obstacles all the way long. Riders will be awarded with scores which is calculated using the number of coins collected and the distance travelled. As, the game distance increases the speed of the game also increases and causes the game to be more challenging. Thus, many interesting items are added in this Space Rider game such as shooting, change of background, obstacles, music and sounds and many more to attract the users.

Game Flow

When the game begins, the rider will be on the land moving forward with a slower speed. The score will increase as the rider move forward. If the user touches the left side of the screen, the rider will shoot out bullets. When the user touches the right side of the screen, the rider will jump. Multiple touches or tap on the screen will maintain the rider position in air. There will be coins heading towards the rider. If the rider collects the coin, the score will increase by 3.

There are 3 types of obstacles that will be heading towards the rider which are crashed UFO, alien and flying UFO. The user must be able to navigate through these obstacles without colliding any of it. If the rider collides with any of the obstacle, the game will be over. However, the rider has the ability to shoot towards the incoming obstacles by pressing the left side of the screen. Shooting and terminating any of those obstacles successfully will increase the score. However, some obstacles require 2bor more shoots to destroy it. Hence, the user must be able to act quickly to destroy the obstacles. Some obstacles will be hiding behind another obstacles which makes the game much more challenging.

Noter

This web application is developed to allow users to write, store and view notes securely. There are many security features included in this web application. One of them is the notes that are stored using this app will be encrypted using BF-CBC cipher, and it can only be decrypted by using the password. The 2-factor authentication feature is also implemented in this application, where it requires the user to enter a password and OTP code that is sent to the user’s registered email when the user signs in to the application. This web application was also developed in the way that it is not prone to Brute Force attacks because the Noter system requires users to use a strong password. This application also includes Captcha to prevent the bots from spamming the website’s sign-in, registration, and password recovery process. The user input will be checked and examined to prevent SQL injection. The Noter web application was built using PHP, HTML, CSS, JavaScript, MySQL database, and Apache server.

My Electrician

My Electrician comes with its main purpose to become a medium for a person or employer who needs help related to electrical works to post a specific task like, wiring, repairing lights, fan installations, and many more so that the electrician could apply for the task that they wished to do from the list. Thus, My Electrician acts as an income-generating system for the tasker and a problem-solving platform for the employers. My Electrician is basically a web-based system with many features like, searching for a task, evaluating tasker, posting new task, applying for a task, and many more. All data that is entered in this application from login credentials up to personal details will be encrypted to secure the data from falling into the wrong hand. Moreover, two-step authentication is also included in this web application to prevent unauthorized access to the system and to keep the user's account more secure. Thus, we believe My Electrician will act as an income-generating system for the tasker and also a problem-solving platform for the employers.

General Flow

The stakeholders of the My Electrician app are the employer, and electrician. The common functionality between employers and electricians is that they both need to register into the application before login and they also can be able to edit or view their personal details. The employer can be able to post a new task by entering all the details related to the task, and they also can edit the details of the task anytime they want. My Electrician app also allows the employer to delete tasks that they have posted. Moreover, the employer also able to approve one of the electricians from the list of electricians who applied for that particular task, and the rest of them will be automatically rejected. Once, the electrician is approved the user has to perform payment, in order to confirm the electrician and once the payment is done the particular task will be removed from the list of available tasks. Besides that, the employer can evaluate the electrician by giving ratings and feedback based on their performance in the task assigned to them. Apart from that, the electricians can search for a specific task that they wish to do and can apply for it by entering their expected fee for the job. The electrician also can filter task based on category, date, time and location.
